# Kern — AI-платформа для строительной индустрии России

## Видение

Kern — вертикально интегрированная экосистема в строительстве.

### Дорожная карта экосистемы

**Kern AI** (сейчас)
SaaS платформа для автоматизации строительных процессов.
Собирает аудиторию, данные о рынке, доверие бренда.

**Kern Materials** (следующий этап)
Онлайн и офлайн магазин строительных материалов.
Клиенты из AI платформы уже знают бренд.

**Kern Build** (долгосрочно)
Строительная компания по возведению частных домов.
Свои материалы + своя смета + свои специалисты.

## Текущий продукт — Kern AI

### Модули
- **AI-сметчик** — загрузка чертежа → смета за 30 секунд ✅
- **Контроль качества** — фото объекта → AI находит дефекты 🔜
- **Генератор документов** — контракты по ГОСТ и СНиП 🔜
- **Тендерная платформа** — подрядчики + AI-оценка заявок 🔜
- **Платформа профессионалов** — верифицированные специалисты 🔜

### Стек
- Next.js 14, TypeScript, Tailwind CSS
- Supabase (база данных, авторизация)
- Gemini 2.0 Flash через OpenRouter (AI)
- Vercel (хостинг)
- Бюджет: $0
